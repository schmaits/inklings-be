GET /api/clubs

Returns an array of all clubs

GET /api/clubs/:clubId

Returns an array with one club

POST /api/clubs

Adds a new club. The following information can be given as a JSON body:

{
name: Club Name (string, required),
summary: Club Summary (string),
members: Club members ([user ID]),
admin: Club Administrator (user ID, required),
currentlyReading: Book that club is currently reading (book ID, required),
read: Books read ([book ID], required)
}

PUT /api/clubs/:clubId/users

Adds a new member to a club. The following information is required as a JSON body:

{
userId: User to be added (user ID, required)
}

PUT /api/clubs/:clubId/currentlyReading

Updates the book the club is currently reading. The following information is required as a JSON body:

{
bookId: Book to be added (book ID, required)
}

PUT /api/clubs/:clubId/read

Adds a book the club has read. The following information is required as a JSON body:

{
bookId: Book to be added (book ID, required)
}

DELETE /api/clubs/:clubId

Deletes a club. The following information is required as a JSON body:

{
userId: Club admin (user ID, required)
}

GET /api/books

Returns an array of all books

GET /api/books/:bookId

Returns an array of one book

POST /api/books/

Adds a new book. The following information can be given as a JSON body:

{
title: Book Title (string, required),
author: Author (string, required),
year: Year originally published (number, required),
coverImageUrl: URL for cover image (string, required),
genres: Book genre(s) ([string], required),
rating: Book rating(s) ([number, min: 0, max: 5])
country: Country of origin (string, required)
}

PUT /api/books/:bookId

Adds a new rating. The following information is required as a JSON body:

{
addedRating: Rating to be added (number between 0 and 5, required)
}

GET /api/users/

Returns an array of all users

GET /api/users/:userId

Returns an array of one user

PUT /api/users/:userId/currentlyReading

Adds or removes a book from the currently reading list. This route requires an update query of 'add' or 'remove', for example:

/api/users/:userId/currentlyReading?update=add

The following information is required as a JSON body:

{
bookId: Book to be added (book ID, required)
}

PUT /api/users/:userId/toRead

Adds or removes a book from the to read list. This route requires an update query of 'add' or 'remove', for example:

/api/users/:userId/toRead?update=add

The following information is required as a JSON body:

{
bookId: Book to be added (book ID, required)
}

PUT /api/users/:userId/booksRead

Adds a book from the books read list. The following information is required as a JSON body:

{
bookId: Book to be added (book ID, required)
}

GET /api/comments/clubs/:clubId

Returns an array of comments for a club

GET /api/comments/clubs/:bookId

Returns an array of comments for a book

POST /api/comments/clubs/:clubId/books/:bookId

Adds a new comment. The following information is required as a JSON body:

{
body: Comment body (string, required)
user: User posting the comment (user ID, required)
}

PUT /api/comments/:commentId

Edits a comment, if the comment was written by the user. The following information is required as a JSON body:

{
updatedBody: Comment body (string, required)
user: User posting the comment (user ID, required)
}

DELETE /api/comments/:commentId/user/:userId

Deletes a comment, if the comment was written by the user.

GET /api/quotes

Returns an array of all quotes

POST /api/quotes/books/:bookId

Adds a new quote. The following information is required as a JSON body:

{
body: Quote body (string, required)
}
